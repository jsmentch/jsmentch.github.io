## Overview

An interactive **p5.js** sketch for visualizing features extracted from naturalistic movie stimuli. I built this as a lightweight browser tool to make complex stimulus annotations more legible for researchers working with movie-based neuroscience data.

This project sits within the broader **Neuroscout** ecosystem and reflects a recurring theme in my work: building tools that make messy, high-dimensional inputs easier to inspect, debug, and communicate. Rather than treating extracted features as opaque model inputs, this visualizer turns them into something explorable and intuitive.

## What it does

- Visualizes **feature time series** extracted from naturalistic stimuli  
- Supports feature sets available through **Neuroscout**  
- Makes it easier to understand what a model is “seeing” at different moments in a movie  
- Helps connect **feature extraction pipelines** to human-interpretable stimulus events  

## Why it matters

This is a small project, but it shows a lot of what I like to do:

- build tools that help people **understand data**  
- connect **research infrastructure** to usable interfaces  
- turn abstract model inputs into something **visual and explorable**  

## My role

Designed and built the visualization in **p5.js**, connected it to the underlying feature structure, and used it as a lightweight interface for inspecting naturalistic stimulus features.
